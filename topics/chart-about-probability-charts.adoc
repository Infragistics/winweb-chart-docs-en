= About Probability Charts

image::Images/Chart_About_Probability_Charts_01.png[The icon used to represent this chart type in the ChartType drop-down in Visual Studio's properties window.]

A  pick:[win-forms=" link:{ApiPlatform}win.ultrawinchart{ApiVersion}~infragistics.ultrachart.shared.styles.charttype.html[probability chart]"]  pick:[asp-net=" link:{ApiPlatform}webui.ultrawebchart{ApiVersion}~infragistics.ultrachart.shared.styles.charttype.html[probability chart]"]  pick:[aspnet-old=" link:{ApiPlatform}webui.ultrawebchart{ApiVersion}~infragistics.ultrachart.shared.styles.charttype.html[probability chart]"]  is a modified link:chart-scatter-chart.html[scatter chart] which is typically used to plot a level of probability (Y) against a variable (X). The major difference between a probability chart and scatter chart is that in a probability chart, the y-axis scale uses a Gaussian distribution (also known as Normal distribution) algorithm.
