= About Radar Charts

image::Images/Chart_About_Radar_Charts_01.png[The icon used to represent this chart type in the ChartType drop-down in Visual Studio's properties window.]

Lines connect all the values in the same series. Radar charts, also called spider charts, are best used to show multiple variables, such as ratings for different areas, performance levels for various divisions or individuals, or progress in a project or other activity. A radar chart compares the aggregate values of a number of data series.
